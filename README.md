Welcome to my personal portfolio repository! This is a modern, responsive personal website built from scratch using vanilla web technologies. It showcases my skills, web development projects, and professional background.

## ✨ Features
- **Pure Vanilla Code:** Built using clean HTML, CSS, and modern JavaScript without heavy frameworks.
- **Fully Responsive:** Optimized for all screen sizes (Desktops, Tablets, and Mobile phones).
- **Smooth Animations:** Interactive UI elements powered by CSS and JS.
- **Project Showcase:** A clean layout displaying my latest work and source code links.
- **Contact Details:** A direct way for recruiters and clients to get in touch.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure and content.
- **CSS3:** Modern styling, flexbox/grid layouts, and responsive media queries.
- **JavaScript (ES6+):** Interactive elements and dynamic UI behavior.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com
   ```
2. Navigate into the project directory:
   ```bash
   cd ahmed_usama_amasha_portfolio
   ```
3. Open the `index.html` file directly in any modern browser.
